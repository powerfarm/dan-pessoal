2\. Configurar um Túnel Seguro com Ngrok
----------------------------------------

**Objetivo:**  
Criar um acesso remoto temporário e seguro para o painel web, ideal para testes ou situações emergenciais.

**Passos:**

*   **Instale o Ngrok:**  
    Se ainda não tiver, faça o download e a instalação do [Ngrok](https://ngrok.com/).
*   **Autentique-se no Ngrok:**  
    Após criar uma conta gratuita (ou paga, se preferir recursos avançados), obtenha seu _authtoken_. No terminal, execute:
    ```bash
    ngrok config add-authtoken SEU_NGROK_AUTHTOKEN
    ```
*   **Crie o Túnel para o Painel Web:**  
    Supondo que o painel web esteja rodando na porta 80 (ou outra porta que você utiliza), execute:
    ```bash
    ngrok http 80
    ```
    O Ngrok exibirá uma URL pública (ex.: `https://abcdefgh.ngrok.io`) que redireciona para o seu NAS.
*   **Exemplo com Python (usando pyngrok):**  
    Se preferir automatizar ou integrar com scripts, instale a biblioteca e execute:
    ```python
    # Instale a biblioteca, se necessário:
    # pip install pyngrok
    from pyngrok import ngrok
    # Configure seu authtoken
    ngrok.set_auth_token("SEU_NGROK_AUTHTOKEN")
    # Inicia um túnel para a porta 80 (ajuste conforme necessário)
    tunnel = ngrok.connect(80, "http")
    print("URL pública do Ngrok:", tunnel.public_url)
    ```
    Salve esse script (por exemplo, como `inicia_ngrok.py`) e execute-o para obter a URL do túnel.
